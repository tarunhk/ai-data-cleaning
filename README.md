#  Data Cleaning & Preprocessing 

##  Project Overview
This project demonstrates how to clean and preprocess raw data before using it for machine learning.  
The Titanic dataset is used to perform data cleaning, handle missing values, encode categorical data, scale features, and detect outliers.

##  Objective
To learn how to prepare raw data for machine learning by performing:
- Data cleaning  
- Handling missing values  
- Encoding categorical variables  
- Feature scaling  
- Outlier detection  

##  Tools & Libraries Used
- Python 3  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## Run the program
- python3 data_cleaning.py

The script will:
1. Load dataset
2. Handle missing values
3. Encode categorical data
4. Normalize features
5. Show outlier boxplot
6. Save cleaned dataset

Output file created:
-cleaned_titanic.csv
